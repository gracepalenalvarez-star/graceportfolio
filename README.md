# Grace Portfolio

A modern portfolio website built with Tailwind CSS.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Build the CSS:
```bash
npm run build
```

3. For development with auto-rebuild:
```bash
npm run watch
```

4. Open `index.html` in your browser to view the website.

## Project Structure

- `index.html` - Main HTML file
- `src/input.css` - Source CSS with Tailwind directives
- `output.css` - Generated CSS (created after running build)
- `tailwind.config.js` - Tailwind configuration
- `package.json` - Node.js dependencies and scripts

## Customization

- Edit `tailwind.config.js` to customize your theme
- Add custom styles in `src/input.css`
- Modify `index.html` to change the content and structure
